Practical Labs — Overview

Three instructor-led labs to practice what you've learned: quick exploration, configuration in Web Tools, and import/dedupe + report run.

Logistics

  • Duration: 3 hours total
  • Work in your assigned Dev Workspace
  • Deliverables: screenshots, logs, and a short summary per lab
Page 51 of 60 • Est. Time: 6 mins

Lab 1 — Quick Exploration & Personalization (35 mins)

Goal: Ensure access and comfort with navigation; create a sample contact and personalize Home.

Step-by-step

  1. Open Siebel App and log in.
  2. Navigate to Accounts → run search: Name contains Test.
  3. Open an Account, About Record → copy Row ID.
  4. Under Contacts related list → New Contact; save and screenshot the form.
  5. Home → Personalize → add KPI tile "My Open Cases" → save.

Deliverable

Single PDF with Row ID, contact screenshot and KPI name.

Est. Time: 35 mins

Lab 2 — Add Field & Expose in UI (70 mins)

Goal: Make a safe configuration change in a Dev Workspace and preview.

Prerequisites

Instructor creates Dev Workspaces for each trainee and assigns permissions.

Steps

  1. Open Web Tools and switch to your Dev Workspace.
  2. Object Explorer → Business Component → Contact. Add field X_PREFERRED_CHANNEL (Text).
  3. Open Contact Applet → add column X_PREFERRED_CHANNEL.
  4. Save to workspace → Deliver to Integration for quick QA preview.
  5. Preview runtime (workspace preview) → create a Contact with the new field set to "Email".

Deliverable

Two screenshots: BC field edit (Web Tools) & runtime UI showing the field. Note workspace name and Row ID.

Est. Time: 70 mins

Lab 3 — Import, Deduplicate & Run Report (75 mins)

Goal: Practice EIM import and deduplication; run a report provided by instructor.

Steps

  1. Download sample_contacts.csv from instructor folder.
  2. Use EIM import (Administration → EIM) with provided import map to load records.
  3. Run duplicate matching (Find Duplicates) using standard rules and merge one pair.
  4. Run instructor-provided report "Accounts by Region" from Run Report menu and download PDF.

Deliverable

Import log text, merge log screenshot, and downloaded PDF report.

Est. Time: 75 mins

Lab Tips & Troubleshooting

If import fails

Check CSV header names, map selection and required fields (no NULL for mandatory fields). Consult logs for row-level errors.

If workspace changes not visible

Ensure you are previewing the correct workspace; clear browser cache; verify delivery completed to Integration.

Est. Time: 8 mins

Q: Import fails on row X

Open import log, find error message, fix CSV, re-run import for failed rows or ask instructor for guidance.
