    Lab 1 ‚Äî Quick Exploration & Personalization (35 mins)

    Goal: Ensure access and comfort with navigation; create a sample contact and personalize Home.

    Step-by-step

    1. Open Siebel App and log in.
    2. Navigate to Accounts ‚Üí run search: Name contains Test.
    3. Open an Account, About Record ‚Üí copy Row ID.
    4. Under Contacts related list ‚Üí New Contact; save and screenshot the form.
    5. Home ‚Üí Personalize ‚Üí add KPI tile "My Open Cases" ‚Üí save.

    Deliverable

    Single PDF with Row ID, contact screenshot and KPI name.

    Est. Time: 35 mins

    Lab 2 ‚Äî Add Field & Expose in UI (70 mins)

    Goal: Make a safe configuration change in a Dev Workspace and preview.

    Prerequisites

    Instructor creates Dev Workspaces for each trainee and assigns permissions.

    Steps

    1. Open Web Tools and switch to your Dev Workspace.
    2. Object Explorer ‚Üí Business Component ‚Üí Contact. Add field X_PREFERRED_CHANNEL (Text).
    3. Open Contact Applet ‚Üí add column X_PREFERRED_CHANNEL.
    4. Save to workspace ‚Üí Deliver to Integration for quick QA preview.
    5. Preview runtime (workspace preview) ‚Üí create a Contact with the new field set to "Email".

    Deliverable

    Two screenshots: BC field edit (Web Tools) & runtime UI showing the field. Note workspace name and Row ID.

    Est. Time: 70 mins

    Lab 3 ‚Äî Import, Deduplicate & Run Report (75 mins)

    Goal: Practice EIM import and deduplication; run a report provided by instructor.

    Steps

    1. Download sample_contacts.csv from instructor folder.
    2. Use EIM import (Administration ‚Üí EIM) with provided import map to load records.
    3. Run duplicate matching (Find Duplicates) using standard rules and merge one pair.
    4. Run instructor-provided report "Accounts by Region" from Run Report menu and download PDF.

    Deliverable

    Import log text, merge log screenshot, and downloaded PDF report.

    Est. Time: 75 mins

    Lab Tips & Troubleshooting

    If import fails

    Check CSV header names, map selection and required fields (no NULL for mandatory fields). Consult logs for row-level errors.

    If workspace changes not visible

    Ensure you are previewing the correct workspace; clear browser cache; verify delivery completed to Integration.

    Est. Time: 8 mins
