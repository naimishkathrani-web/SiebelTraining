Your Assignment: Corporate Billing Trends

Report Configuration

  • Name: TRAINEE02_Billing_Dispute_Analysis
  • Data Source: Service Request BC (Billing Disputes)
  • Columns: SR Number, Account, Dispute Amount, Reason, Status, Created Date
  • Filter: Type = "Billing Dispute" AND Created Date = Last 90 Days
  • Group By: Dispute Reason
  • Calculations: COUNT(SRs), SUM(Dispute Amount), AVG(Resolution Days)
