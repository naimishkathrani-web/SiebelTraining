5) Troubleshooting

Common Issues & Fixes

  • Cannot reach URLs: Verify corporate proxy/VPN rules. Try another network if possible.
  • Login fails: Check username (TRAINEE01..), case-sensitive password (WELCOME123), and keyboard layout.
  • Blank page or errors: Hard refresh (Ctrl+F5); clear cache for the site; disable extensions temporarily.
