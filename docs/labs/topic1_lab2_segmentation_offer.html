Goal

Prerequisites

Steps

  3. Criteria examples:

  4. Execute/Preview results; ensure at least a few records appear.

  6. Link the Offer note to the Segment (or keep as documentation in the record).

Validation

  • Segment or Saved Query exists with the specified name
  • Criteria reflect prepaid and at-risk behavior
  • Offer note documented with SMS text

Troubleshooting

  • No Marketing screen? Use Contacts list and Saved Query instead.
  • Missing usage fields? Use Last Recharge Date or Last SR Date as proxy.
  • Zero results? Relax criteria (e.g., 60 days) and re-run.
