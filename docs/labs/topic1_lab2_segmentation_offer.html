Goal

Learn how to set up product catalog, create new products, attach pricelists, and activate products for sale.

Prerequisites

  • Access to Product Administration screens
  • Basic understanding of telecom product types (plans, bundles, add-ons)

Exercise 1: Navigate to Product Administration

Access Product Catalog

  1. Click Site Map button
  2. Expand Product & Catalog section (or Administration ‚Üí Products)
  3. Click Products ‚Üí Opens product list view
  4. Observe existing products: Product Name, Type, Status, Price List

‚úÖ Success Check: You should see the product catalog with existing telecom products.

Exercise 2: Create a New Product

Add 5G Unlimited Plan

  1. In Products list view, click New button
  2. Fill in required fields:
    • Product Name: 5G Plan - Unlimited Data
    • Product Type: Select Service Plan (or Subscription)
    • Description: Unlimited 5G data with unlimited calls and SMS
    • Status: Draft (not active yet)
    • Part Number/SKU: 5G-UNLIM-001
  3. Click Save ‚Üí Product record is created
  4. Note the Product # generated by the system

‚úÖ Success Check: New product appears in list with Status = "Draft".

Exercise 3: Define Product Attributes

Add Product Details

  1. With your new product open, navigate to Characteristics or Attributes tab
  2. Add product attributes (if available in your environment):
    • Data Allowance: Unlimited
    • Voice Minutes: Unlimited
    • SMS Count: Unlimited
    • Network Type: 5G
    • Contract Duration: 30 Days (monthly)
  3. If Attributes tab is not available, add this information in the Description field
  4. Save changes

Exercise 4: Create Price List

Set Product Pricing

  1. In Product record, navigate to Price Lists applet or tab
  2. Click New to create a new price list item
  3. Fill in pricing details:
    • Price List Name: Standard Retail Pricing (or select existing)
    • List Price: 50000 (TZS 50,000 monthly)
    • Currency: TZS (Tanzanian Shilling)
    • Effective Start Date: Today's date
    • Effective End Date: Leave blank (no expiration)
  4. Save the price list entry

üí° Note: Price lists control pricing for different customer segments (retail, wholesale, corporate).

Exercise 5: Attach Price List to Product

Link Pricing

  1. Still in the Product record, verify Price Lists section shows your new price
  2. If not automatically linked:
    • Click Add in Price Lists applet
    • Select Standard Retail Pricing
    • Enter price: 50000 TZS
    • Save
  3. Verify price is now visible on product

‚úÖ Success Check: Product shows price of 50,000 TZS under Standard Retail Pricing.

Exercise 6: Activate the Product

Make Product Available for Sale

  1. Return to the main Product form view
  2. Locate the Status field (currently "Draft")
  3. Change Status from Draft to Active
  4. Save the product
  5. Verify product status is now "Active" in the product list

‚úÖ Success Check: Product is now available for selection in Quotes and Orders.

Try navigating to Quotes ‚Üí New Quote ‚Üí Add Product ‚Üí Search for "5G Plan" to verify.
