Exercise 1: Create Contact as Lead

Add New Lead Contact

  1. Click Site Map ‚Üí Contacts ‚Üí Opens contact list view
  2. Click New button to create new contact
  3. Fill in contact information:
    • First Name: Amina
    • Last Name: Juma
    • Email: amina.juma@example.com
    • Phone: +255 712 345 678
    • Contact Type: Select Lead (very important!)
    • Lead Source: Web or Referral
  4. Click Save
  5. Note the Contact # generated

‚úÖ Success Check: New contact created with Type = "Lead".

Exercise 2: Create Opportunity for Contact

Add Sales Opportunity

  1. While still in Amina Juma's contact record, locate the Opportunities applet (usually in lower section)
  2. Click New in the Opportunities applet
  3. Fill opportunity details:
    • Opportunity Name: 5G Plan Upgrade - Amina Juma
    • Revenue: 600000 (TZS 600,000 annual value)
    • Close Date: Select date 30 days from today
    • Probability: 50%
    • Status: Prospecting or Open
    • Sales Stage: Qualification
  4. Click Save
  5. Note the Opportunity # generated

‚úÖ Success Check: Opportunity is linked to Amina Juma contact.

Exercise 3: Associate Opportunity with Account

Link to Customer Account

  1. Navigate to Site Map ‚Üí Opportunities ‚Üí Find your newly created opportunity
  2. Open the opportunity record
  3. In the opportunity header, locate the Account field
  4. Option A - Link to Existing Account:
    • Click the lookup icon (magnifying glass) next to Account field
    • Search for and select an existing test account
    • Click OK
  5. Option B - Create New Account:
    • Click New button in the Account lookup
    • Enter Account Name: Juma Family Account
    • Account Type: Consumer
    • Save the account
  6. Save the opportunity with linked account

üí° Note: Linking to an Account ensures all transactions (orders, billing) are properly associated.

Exercise 4: Create Quote for Opportunity

Generate Quote with Products

  1. In the Opportunity record, locate the Quotes applet (lower section)
  2. Click New in Quotes applet
  3. Fill quote header:
    • Quote Name: 5G Plan Quote - Amina
    • Status: Draft
    • Valid From: Today's date
    • Valid To: 14 days from today
  4. Save the quote
  5. Now add product line items - Click Products tab or Line Items applet
  6. Click New or Add Product
  7. Search for product: 5G Plan (created in Lab 2)
  8. Select the product and fill:
    • Quantity: 1
    • Price: 50000 TZS (should auto-populate from price list)
    • Duration: 12 months
  9. Save the line item
  10. Verify Quote Total = 600,000 TZS (50,000 √ó 12 months)

‚úÖ Success Check: Quote contains at least one product line with correct pricing.

Exercise 5: Convert Quote to Order

Submit and Generate Order

  1. In the Quote record, locate the Status field
  2. Change Status from Draft to Submitted (or Accepted)
  3. Save the quote
  4. Look for Convert to Order button (or similar action button)
  5. If button exists: Click Convert to Order ‚Üí System automatically creates order
  6. If no button: Manually create order:
    • Navigate to Site Map ‚Üí Orders
    • Click New
    • Fill Order fields:
      • Account: Juma Family Account
      • Contact: Amina Juma
      • Order Type: New Service
      • Status: In Progress
    • Add order line items matching quote products
    • In Order header, reference the Quote # in notes or description
  7. Save the order

‚úÖ Success Check: Order record created and linked to Quote/Opportunity/Account.

Exercise 6: Verify Order Created

Validate Complete Flow

  1. Navigate to Site Map ‚Üí Orders ‚Üí View all orders
  2. Find your newly created order (sort by Created date)
  3. Open the order and verify:
    • Order #: System-generated number
    • Account: Juma Family Account
    • Contact: Amina Juma
    • Status: In Progress (or Submitted)
    • Order Lines: At least one product (5G Plan - Unlimited Data)
    • Total Amount: 600,000 TZS
  4. Optional: Update Order Status to Released or Completed to simulate fulfillment
