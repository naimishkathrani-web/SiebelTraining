⚡ Lab 3: Capture Order and Walk Activation

Estimated time: 15–20 minutes

Goal

Create a prepaid bundle order, submit it, and walk through status updates to simulate activation and customer notification.

Prerequisites

  • Login: TRAINEE01–10 / Welcome123
  • Product catalog contains at least one prepaid bundle product (or use a placeholder)

Steps

  1. Open Site Map → Orders → All Orders (List).
  2. New Order: Set Account/Contact, Order Type = Add-on, Status = In Progress.
  3. Order Line: Add product (e.g., "Prepaid 2GB Bundle"). Quantity = 1.
  4. Submit: Click Submit (or set Status to Submitted) and note Order #.
  5. Simulate: Update Status sequence to Released → In Provisioning → Complete.
  6. Assets: Verify a new Asset was created/updated for the subscriber (if available).
  7. Notify: Add an Activity "Notify customer" with channel SMS/Email.

Validation

  • Order header and line saved with correct product
  • Status progressed through key milestones
  • Asset created/updated (if feature enabled)

Troubleshooting

  • Can't find product? Use any available catalog item or note a placeholder in the line
  • Submit button missing? Manually set Status fields to simulate flow
  • No Assets? Skip asset validation and complete with Status = Complete
