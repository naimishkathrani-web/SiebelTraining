Duplicates degrade data quality. Siebel provides tools to identify and merge duplicate records.

Identifying Duplicates

Duplicate Detection Strategies

  • Exact Match: Same Name + Email + Phone
  • Fuzzy Match: Similar names (soundex, edit distance)
  • Rule-Based: Business logic (same company + similar name)

Merge Process

  1. Identify duplicate records
  2. Select master record (most complete/recent)
  3. Merge related records (activities, opportunities)
  4. Deactivate or delete duplicate
