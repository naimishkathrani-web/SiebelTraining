⚙️ 6. Creating Segments in Siebel

Sub-topic 6 of 8
75% Complete

Hands-on: Build dynamic and static segments using Siebel Marketing.

Segment Types

Dynamic Segments

Membership recalculated automatically based on criteria

Example: "High ARPU Customers" → ARPU > $50

Use when: Criteria change frequently; need real-time updates

Static Segments

Fixed list of members; manual updates required

Example: "Conference Attendees 2024"

Use when: One-time campaign; historical snapshot needed

Creating a Dynamic Segment

  1. Navigate to Marketing → Segments
  2. Click New Segment
  3. Set Type = "Dynamic"
  4. Define criteria: [ARPU] > 50 AND [Status] = 'Active'
  5. Save and run query
