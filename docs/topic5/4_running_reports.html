Running Reports

Report Execution Workflow

  1. Select Report: Navigate to Reports → Choose "Opportunity Pipeline"
  2. Set Parameters: Date range (Q1 2024), Territory (Western Region), Minimum Revenue ($10K)
  3. Run Report: Click "Execute" button
  4. View Results: Report displays in browser (500 opportunities found)
  5. Interact: Sort, filter, drill-down, export

Report Parameters

Common Filters

  • Date Range: Last 30 days, This Quarter, Custom (01/01/2024 - 03/31/2024)
  • Owner: Current User, My Team, All Users, Specific Rep (select from dropdown)
  • Status: Open, Closed Won, Closed Lost, All
  • Territory: Northeast, Southeast, Midwest, West, All Territories
  • Threshold: Revenue > $50K, Probability > 50%

Advanced Filters

  • Multi-Select: Industry IN (Telecom, Technology, Healthcare)
  • Wildcards: Account Name LIKE "%Corp%"
  • Date Math: Close Date BETWEEN (Today AND Today + 30 days)
  • AND/OR Logic: (Stage = 'Negotiation' OR Probability > 75%) AND Revenue > $100K

Viewing Results

Table View (Default)

  • Columns: Account, Opportunity, Revenue, Close Date, Stage, Owner
  • Sorting: Click column header to sort (ascending/descending)
  • Paging: 50 rows per page, navigate with Previous/Next buttons
  • Totals: Footer row shows Sum of Revenue, Count of Opportunities

Chart View

  • Bar Chart: Revenue by Sales Stage (visual pipeline)
  • Pie Chart: Win/Loss distribution (60% Won, 40% Lost)
  • Line Chart: Revenue trend over time (monthly)
  • Interactive: Hover for values, click slice to drill-down

Summary View

  • Grouped Data: Opportunities grouped by Territory
    • Northeast: 150 opportunities, $5.2M total revenue
    • Southeast: 120 opportunities, $4.1M total revenue
    • West: 230 opportunities, $8.9M total revenue
  • Expand/Collapse: Click (+) to expand group, see individual records

Interactive Features

Drill-Down

Example:
  • Report shows "Western Region: $8.9M revenue"
  • Click region → Drill to see individual opportunities in West
  • Click opportunity "Acme Corp 5G Upgrade" → Open full opportunity record

Dynamic Filtering

  • Post-Execution Filter: Report shows 500 rows → Type "HealthCorp" in search box → View narrows to 3 matching rows
  • Column Filter: Click funnel icon on Revenue column → Show only Revenue > $100K

Sorting & Ranking

  • Sort by Revenue (descending) → See top deals first
  • Multi-level sort: Sort by Territory (ascending), then Revenue (descending)

Output Formats

1. HTML (Browser)

  • Default view, interactive, real-time filtering
  • Best for on-screen analysis, drill-down

2. Excel (.xlsx)

  • Export → Excel → Opens in Microsoft Excel or Google Sheets
  • All rows exported (not just current page)
  • Retains formatting (bold headers, totals row)
  • Use for: Pivot tables, offline analysis, email distribution

3. PDF

  • Export → PDF → Generates formatted document
  • Includes: Company logo, report title, filters used, generation timestamp
  • Use for: Executive presentations, printing, archiving

4. CSV (Comma-Separated Values)

  • Raw data, no formatting
  • Use for: Importing to other systems, data warehouse loads

Saving & Sharing

  • Save Filters: Save filter criteria as "My Q1 Pipeline View" → Reload anytime
  • Bookmark: Save report URL for quick access
  • Email: Export → Email → Recipients receive PDF attachment
  • Share Report Definition: Grant access to other users ("Sales Team can run this report")

Example: Running Sales Pipeline Report

Scenario: Sales Manager wants to review team's pipeline
  1. 9:00 AM: Navigate to Reports → Sales Reports → "Opportunity Pipeline"
  2. 9:01 AM: Set filters: Date = This Quarter, Owner = My Team (5 reps), Stage ≠ Closed Lost
  3. 9:02 AM: Click "Run" → 147 opportunities displayed (Total: $4.2M weighted revenue)
  4. 9:03 AM: Sort by Close Date (ascending) → See deals closing soon first
  5. 9:05 AM: Click "Negotiation" stage bar in chart → Drill to 23 negotiation deals
  6. 9:10 AM: Export to Excel → Send to VP Sales for review
